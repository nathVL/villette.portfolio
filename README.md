# Personal Portfolio

Personal portfolio of **Nathan Villette** built with Next.js.

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nathVL/villette.dev.git
   cd villette.dev
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Go to [http://localhost:3000](http://localhost:3000)**
